#	INSTALLATION

1) Use Python:

Download the code:
	
	git clone https://github.com/odioski/FWCracker_v3.git

Get PySerial:
	
	pip install pyserial

Run FWCracker:

	python -m FWCracker.py

2) Launch the Win exe:

Navigate to dist/FWCracker, inside is FWCracker.exe

3) Install FWCracker, 

Double-click the FWCracker_v3_Installer located in /INSTALLER

Launch FWCracker


# SUPPORT

If pyserial-ports wont launch you can add it to your *PATH* or find your port in the Device Manager.
From there just input the port and launch.
